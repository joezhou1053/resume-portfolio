4. keras   的 ValueError: Unable to import backend : mxnet

这个错误是因为你的环境中安装了独立版本的Keras，并且配置为使用MXNet后端，但你现在想要使用TensorFlow。有几种方法可以解决这个问题：
设置环境变量（推荐）：
在导入TensorFlow之前，先设置Keras后端为TensorFlow：


import os
os.environ['KERAS_BACKEND'] = 'tensorflow'  # 必须在导入keras前设置

import tensorflow as tf
from tensorflow.keras import Model
from tensorflow.keras.layers import Input, Dense, Dropout, LSTM
