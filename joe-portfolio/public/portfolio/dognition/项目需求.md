# Dognition Project Requirements

## 项目概述

**任务目标**: 为 Dognition 公司进行数据驱动的业务流程改进提案

**公司背景**
- Dognition 是一家帮助您与狗狗建立更深层连接的公司
- 通过前所未有的视角，让您深入了解狗狗的个性和能力

## 核心任务

公司委托我们帮助找出可以通过哪些业务变革来**增加用户在网站上完成的测试数量**。

## 产出要求

### 数据驱动的业务流程改进提案

### 演示形式要求

1. **开场陈述** (60秒或更短)
   - 概述问题和/或解决方案

2. **建议部分**
   - 至少1个（最多3个）基于数据分析的建议

3. **可视化支持**
   - 至少2个（最多5个）使用 Tableau 制作的图表/仪表板
   - 必须直接支持您的建议
   - 其中至少1个是条形图或折线图（来自第2周或第3周的练习）

4. **结论陈述** (60秒或更短)
   - 重申建议的重要性和动机
   - 配合分析和可视化支持

5. **展示方式**
   - 使用 Tableau 的 Story 功能展示支持建议的数据

## 评分标准

提交作品将从以下方面进行评估：

1. **开场和结论的质量**
2. **图表和可视化的清晰度**
3. **数据故事的逻辑性**
4. **建议的可行性和实用性**
5. **演示设计和表达**
